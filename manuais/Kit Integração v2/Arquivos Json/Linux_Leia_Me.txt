Pacotes que devem ser instalados para rodar no Ubuntu:
---
sudo apt install pcscd libpcsclite1
sudo apt install pcsc-tools
sudo apt install libccid
sudo apt install python3-pyscard
---


Antes de executar a primeira vez, certificar que o arquivo list_readers.py é executável
---
~ # chmod +x list_readers.py
---


Para rodar, é necessário alterar o provedor TLS, exemplo:
---
~ # MONO_TLS_PROVIDER=legacy mono IPC.eFRETE.ValePedagio.exe
---
